# 95 Solutions

Reference solutions for [95](https://95ninefive.dev) challenges.

---

## Want to contribute?

We'd love to see **your** approach. Different languages, different styles — all welcome.

### Rules to submit a solution:

> **You must complete the challenge first.**
>
> Include a **screenshot** showing all stages passed on the platform. 

### How to contribute:

1. Fork this repo
2. Add your solution under the challenge folder
3. Attach your **completion screenshot** in the PR
4. Open a pull request
